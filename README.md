# Eテレさがすくん 🔍

NHK Eテレ「おかあさんといっしょ」「いないいないばぁ」の録画から、
**「あの曲どこにある？」を最速で見つけるための索引アプリ**。

## なぜ作ったか

2歳児のいる家庭あるある：
- 子どもが突然「あれ見たい！」と叫ぶ
- 親はHDDレコーダーを早送り早送り…で探す
- 子はその間ギャン泣き、地獄

→ 直近2週間の放送内容を曲・コーナー単位でDB化し、
　「日付・放送順」を一目で確認できるようにする。

## 仕組み

```
[ tokyofukubukuro.com 週報 ]   ← 毎週更新の現役ファンサイト
        ↓ WordPress REST API
[ scraper.py ]                ← Python、構造化HTMLからメタデータ抽出
        ↓ data.json
[ index.html ]                ← 単一HTMLのフロントエンド
```

- **索引のみ**：レコーダー操作はユーザー（DTCP-IPでサードパーティ制御不能のため）
- **直近2週間ローリング**：家庭用レコーダーの容量制約に合わせる
- **完全自動・追加ハードなし**：PCだけで完結

## 使い方

```bash
# データ取得（直近2週間）
python3 -m pip install --user beautifulsoup4 requests
python3 scraper.py

# プレビュー
python3 -m http.server 8765
# → http://localhost:8765/
```

## 機能

- 📅 日別タイムライン表示（放送順に番号付き）
- 🔍 曲名・コーナー名検索（カナ↔ひらがな自動同一視）
- ⭐ お気に入り登録（localStorage）
- 🎨 コーナー種別の色分け（うた／人形劇／アニメ／体操／コーナー）
- 🎵 ネスト曲対応（コーナー内で歌われる曲も別エントリーで抽出）
